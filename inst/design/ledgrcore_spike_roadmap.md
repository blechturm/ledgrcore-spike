# ledgrcore-spike Roadmap

**Status:** Active roadmap.
**Authority:** Milestone sequence and current planning horizon.

## Vision

`ledgrcore-spike` is a sister research repository for deciding whether ledgr's
compiled fold-core path should be built at all.

## Milestone Sequence

| Milestone | Status | Goal | Authoritative record |
| --- | --- | --- | --- |
| M1 | Planned | Measurement spike complete. Two compiled fold cores (Rust via extendr, C++ via cpp11) measured against post-v0.1.8.10 production R on the four load-bearing numbers per the K1 charter. Decision-rule verdict authored: build authorized, scope further, or park. | Future measurement spec |

No further milestones are planned here. If the verdict is build authorized, this
repository may graduate to a real `ledgrcore` roadmap. If not, the work parks.
